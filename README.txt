## About

Aptech Benin is a premier institution offering cutting-edge education in information technology and related fields. Established with a vision to empower individuals with technical skills and knowledge, Aptech Benin provides a range of courses designed to meet global standards, ensuring students are equipped with the latest industry-relevant skills.

## Features

- **Homepage:** Featuring a hero section with a background video, introductory information, and call-to-action buttons.
- **Courses:** Detailed descriptions of various courses offered by Aptech Benin.
- **Faculty:** Information about the experienced faculty members.
- **Partners:** Showcase of Aptech partners with logos and names in a responsive table.
- **Events and Blogs:** Latest news, articles, and events with "Read More" functionality for expanded content.
- **Contact:** Embedded Google Maps with Aptech Benin's location.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap**
- **Google Maps API (for location)**
- **FlexSlider (for hero section)**
